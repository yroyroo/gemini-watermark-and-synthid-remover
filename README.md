# Gemini Watermark & SynthID Remover

A unified C++ tool for removing watermarks from images and videos generated by Google Gemini and Google Veo.

**Three watermark types are in scope:**

| Watermark | Type | Where | Visibility |
|-----------|------|-------|------------|
| Gemini sparkle logo | Alpha-blended overlay | Gemini-generated images | Visible (semi-transparent) |
| SynthID | Spread-spectrum Fourier signal | Gemini-generated images | Invisible |
| Veo text watermark | Alpha-blended text overlay | Veo-generated videos | Visible |

## Project Status

**Early development.** Drawing from two existing open-source tools, extending and combining their approaches into a unified C++ toolkit with a CLI-first architecture.

## Architecture

**Language:** C++20 — chosen for performance-critical pixel/FFT operations, direct memory access, and the ability to ship as a single static binary with zero runtime dependencies.

**Approach: CLI-first, GUI later.** The core engine is developed and tested entirely through the command line. This keeps the iteration loop fast (compile, run, check output) and ensures every code path is testable and scriptable from day one. A GUI can be added as a thin layer on top once the engine is solid.

**Planned project structure:**

```
src/
├── core/                   # Engine library (CLI + GUI shared)
│   ├── visible_watermark/  # Alpha blending, detection, inpainting
│   ├── synthid/            # Spectral analysis, codebook, FFT subtraction
│   └── video/              # Frame extraction, Veo watermark processing
├── cli/                    # Command-line interface
└── gui/                    # Future: desktop GUI (ImGui)
```

## Technical Background

### Visible Watermark Removal (Gemini & Veo)

Gemini and Veo apply visible watermarks using **alpha blending**:

```
watermarked = alpha * logo + (1 - alpha) * original
```

Removal reverses the blending equation:

```
original = (watermarked - alpha * logo) / (1 - alpha)
```

This is a deterministic, mathematically exact operation when the alpha map is known and the image hasn't been resized or recompressed. Residual artifacts from post-processing are cleaned up using inpainting (Navier-Stokes, TELEA, or soft Gaussian blend) and optionally AI-based denoising.

Gemini uses two watermark sizes:
- **48x48** for images <= 1024px on either dimension
- **96x96** for images > 1024px on both dimensions

Both are placed in the bottom-right corner.

### SynthID Removal (Invisible Watermark)

SynthID is Google DeepMind's invisible watermarking technology. Unlike visible watermarks, it is embedded **during image generation** using a spread-spectrum technique in the Fourier domain:

- **Carrier frequencies** carry the watermark signal at specific bins in the FFT spectrum
- Carrier positions are **resolution-dependent** (different positions for different image sizes)
- The watermark's **phase template is identical** across all images from the same model (>99.5% cross-image coherence)
- The **green channel** carries the strongest signal
- A neural encoder adds a learned noise pattern modulated at these carrier frequencies

Removal works via **spectral codebook subtraction**:

1. Build per-resolution watermark fingerprints (carrier positions, magnitudes, phases) from reference images
2. Compute FFT of the input image
3. At each carrier bin, subtract the estimated watermark energy weighted by confidence and channel strength
4. Inverse FFT to recover the cleaned image

This achieves ~75% carrier energy drop and ~91% phase coherence drop while maintaining 43+ dB PSNR (high image quality).

## Source Projects

This project builds on the research and techniques from:

### [reverse-SynthID](https://github.com/aloshdenny/reverse-SynthID) — by aloshdenny

SynthID invisible watermark detection and removal via spectral analysis.

- Pure signal processing (no GPU or deep learning frameworks required)
- V3 Multi-Resolution Spectral Codebook Subtraction
- Multi-scale, multi-denoiser Bayesian detection pipeline (~90% accuracy)
- Resolution-dependent carrier frequency discovery and profiling

### [GeminiWatermarkTool](https://github.com/allenk/GeminiWatermarkTool) — by Allen Kuo (allenk)

Gemini visible watermark removal via reverse alpha blending (C++20).

- Reverse alpha blending with calibrated 48x48 and 96x96 alpha masks
- Three-stage NCC watermark detection (spatial, gradient, variance)
- AI-assisted residual cleanup (FDnCNN neural network via NCNN + Vulkan)
- Software inpainting fallback (NS, TELEA, Soft Inpaint)
- Also powers [VeoWatermarkRemover](https://github.com/allenk/VeoWatermarkRemover) for video watermark removal
